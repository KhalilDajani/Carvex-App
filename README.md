# 🚗 Carvex - Car Marketplace Mobile Application

Carvex is a mobile application built with Flutter that allows users to buy and sell cars in a simple, fast, and modern way. The app connects buyers and sellers and provides a smooth user experience with real-time data and clean UI.

## 📱 Features

- Browse available cars for sale
- Post new car listings
- Save favorite cars
- User authentication (login & register)
- Upload car images
- Search and filter cars
- Chat with sellers (if implemented)
- Location-based listings (if available)

## 🛠️ Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- State Management (Provider / setState / Bloc if used)

## 📂 Project Structure

lib/

- widgets/
- services/
- providers/
- main.dart

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed
- Android Studio or VS Code
- Firebase project (if used)

### Installation

1. Clone the repository:
git clone https://github.com/KhalilDajani/Carvex-App.git

2. Go to project directory:
cd carvex

3. Install dependencies:
flutter pub get

4. Run the app:
flutter run

## 🔥 Firebase Setup (if used)

- Create a Firebase project
- Add Android/iOS apps
- Download google-services.json and place it in:
android/app/
- Enable Authentication, Firestore, and Storage
